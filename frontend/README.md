# Amazon Connect Multi-Modal Frontend

Simple vanilla JavaScript frontend for Amazon Connect chat and voice demo.

## Setup

```bash
npm install
```

## Development

```bash
npm run dev
```

This will start a local dev server at http://localhost:5173

## Build

```bash
npm run build
```

This creates a production build in the `dist` folder.

## Configuration

Update `config.js` with your:
- AWS region
- Connect instance ID
- Connect contact flow ID
- API Gateway endpoint
